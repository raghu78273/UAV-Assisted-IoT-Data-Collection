# UAV-Assisted IoT Data Collection

## Overview
This project presents a **UAV-assisted IoT data collection framework** 
designed for infrastructure-free environments where traditional network 
connectivity is unavailable. UAVs act as mobile data collectors, visiting 
IoT sensor nodes and gathering data using **Delay-Tolerant Networking (DTN)** 
based communication protocols.

## Key Algorithms Implemented
| Algorithm | Description |
|-----------|-------------|
| Energy-Aware Replanning | UAV dynamically replans path based on remaining battery |
| Multi-UAV Coordinated Coverage | Multiple UAVs coordinate to cover all sensor nodes efficiently |
| ML-based Node Prioritization | Machine learning used to prioritize high-value sensor nodes |

## Features
- Browser-based UAV path planning simulation using HTML5 and JavaScript
- DTN-based communication for delay-tolerant data transfer
- Adaptive path planning to maximize data collection efficiency
- Comparative analysis across multiple routing scenarios
- Supports both single and multi-UAV configurations
